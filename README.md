## About
基于 python + selenium 实现自动登入脸书，后续可结合BeautifulSoup采集粉丝团贴文与留言

## Feature

* 引用 selenium 库, 实现自动登入脸书模拟用户操作
* 后续可扩展爬虫应用

## Requires
Python 3.11.0  
selenium 4.8.0  
webdriver_manager 3.8.5   
configparser 5.3.0  

## Usage
```
python main.py
```
